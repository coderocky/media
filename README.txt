The project contains 3 python files 
1. media.py - Contains the definition for Class Movie. The movie class has attributes like tilte, story line, movie poster and you tube trailer url.
2. entertainment_center.py - Contains the definition for the movie instances. Makes a list of all the different movie instances and uses the fresh_tomatoes.py for display and presentation of webpage.
3. fresh_tomatoes.py - Used to open up the url for displaying all the movies and playing trailers. 

To run the program, either
1. Open the fresh_tomatoes.html or
2. double click on entertainment_center.py or 
3. open the entertainment_center.py using IDLE and click on Run --> Run Module.
